# team16-co2026
Some submissions from Team16 for the GUTS Code Olympics 2026

## Pigeon

Submission for Nina's Bird Challenge.

Compilation:
```g++ pigeon/main.cpp -o pigeon -lncursesw```

Usage:

Run
```./pigeon.exe```

Use arrow keys to cycle through options, enter to confirm. Exit using ctrl+c or q. Enjoy the storyline!

## Much Ado about Morse Code

Submission for [SAS] Morse LOLCoder.

Written in SPL (Shakespeare Programming Language).

Use [this online interpreter](https://esolangpark.vercel.app/ide/shakespeare), just copy the source code, give an input string (only capital letters, numbers, spaces, and any of the following symbols: , . : ; " ( ) ').

Outputs a Morse code representation of the input string, with "/" for spaces.
